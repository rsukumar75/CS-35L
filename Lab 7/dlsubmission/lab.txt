access("/etc/ld.so.preload", R_OK)      = -1 ENOENT (No such file or directory)
open("/etc/ld.so.cache", O_RDONLY|O_CLOEXEC) = 3

The following system calls load libc.so.6:

open("/lib64/libc.so.6", O_RDONLY|O_CLOEXEC) = 3
read(3, "\177ELF\2\1\1\3\0\0\0\0\0\0\0\0\3\0>\0\1\0\0\0\340$\2\0\0\0\0\0"..., 832) = 832

The following system call writes the result:

write(1, "-0.16055653857469052", 20-0.16055653857469052)    = 20

Thus, the access, open, read, and write system calls are related to dynamic linking.
